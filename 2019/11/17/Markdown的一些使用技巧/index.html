注意，如果标题有空格等其他符号，需要在转成-符号，括号中只允许字母、汉字、数字和-。

嵌套列表有以下两种实现方式。

  1. 列表前插入四个空格（一个TAB），这样的方式比较简单，每次按一次TAB键就可以。
  2. 列表前插入对齐的空格数，不固定每次嵌套所需的空格数量，比如每个嵌套都插入两个空格，只要列表对齐就行。
    在typora上写markdown可以使用非strict模式，可以不用那么规范，好像会自动调整空格数。
